Vous avez un PDF de 80 pages à lire pour demain matin ? ChatGPT peut vous en extraire l'essentiel en moins de 2 minutes. Voici exactement comment faire, avec les méthodes gratuites et payantes.

## Méthode 1 — ChatGPT Plus avec upload direct (la plus simple)

Avec un abonnement ChatGPT Plus (20$/mois), vous pouvez uploader directement votre PDF dans la conversation.

**Étapes :**
1. Ouvrez chat.openai.com
2. Cliquez sur l'icône trombone pour joindre votre fichier
3. Sélectionnez votre PDF (jusqu'à 512 Mo)
4. Utilisez ce prompt :
```
Résume ce document en 5 points clés maximum.
Pour chaque point, indique la page source.
Utilise un langage professionnel adapté à un dirigeant d'entreprise.
```

**Résultat attendu :** Un résumé structuré en moins de 30 secondes, avec les références aux pages du document original.

## Méthode 2 — Version gratuite (copier-coller du texte)

Sans abonnement, copiez simplement le texte de votre PDF et collez-le dans ChatGPT.

**Limite :** ChatGPT accepte environ 3 000 mots par message. Pour les documents longs, découpez-les en sections.

**Le prompt pour les documents longs :**
```
Voici la section [1/3] d'un document.
Ne résume pas encore, retiens uniquement les informations clés.
Je t'enverrai les sections suivantes avant de demander le résumé final.

[Collez votre texte ici]
```

## Les meilleurs prompts selon le type de document

**Pour un rapport financier :**
```
Résume ce rapport en identifiant :
- Les 3 chiffres clés à retenir
- Les risques mentionnés
- Les recommandations principales
```

**Pour un contrat :**
```
Analyse ce contrat et identifie :
- Les obligations de chaque partie
- Les dates importantes
- Les clauses inhabituelles ou à risque
```
